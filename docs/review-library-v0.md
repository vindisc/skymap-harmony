# Review Library v0 产品需求文档

版本：v0.1

状态：Draft

所属产品：Skymap Product

适用客户端：Mac Client、Harmony Client

## 背景

Skymap 已形成「拍摄 -> 摄影复盘 -> review.json -> 跨端同步」链路，但复盘数据仍以文件形式存在。Review Library 用于把复盘记录沉淀为可查看、可搜索、可筛选的长期数据资产。

## v0 目标

- 查看所有摄影复盘记录。
- 搜索标题、画面事实、核心关系、延伸理解、当前卡点。
- 按是否成立、卡点、复盘人筛选。
- 进入详情查看复盘记录。

## v0 非目标

不做 AI 分析、成长统计、图表分析、自动评分、云同步、多用户。

## 双端职责

- Mac：分析工作台，支持查看、搜索、筛选和通过现有复盘卡工作流编辑。
- Harmony：移动记录器，支持查看、搜索、筛选，暂不支持复杂编辑。

## 数据来源

核心实体来源为 `ReviewExchangeJSON`，即 `review.json`。Harmony 本地使用 `ReviewCardHistoryService` 保存复盘历史，并在历史条目里保留 `reviewerText`，用于复盘库筛选。

旧历史记录如果没有 `reviewerText`，载入时会归一为空复盘人；这些记录仍会出现在「全部复盘人」下，但不会出现在具体复盘人筛选项里，也不会匹配某个具体复盘人筛选。

## Harmony v0 页面

首页入口为「复盘库」。复盘库页面包含：

- 搜索框。
- 是否成立筛选。
- 当前卡点筛选。
- 复盘人筛选。
- 复盘记录列表。
- 空状态和无匹配结果状态。

## 成功标准

用户拥有 50+ 复盘记录时，仍能快速找到目标复盘，无需依赖文件夹搜索。

## 删除边界

复盘库删除只删除本地历史记录，不删除原照片文件，不删除已导出的复盘图片，也不删除用户已经导出的原始 `review.json`。

## 后续版本

Review Library v1 轻量体验增强见 [review-library-v1.md](./review-library-v1.md)。标签、收藏、批注仍属于后续路线，不进入 v1 本轮轻量增强。
