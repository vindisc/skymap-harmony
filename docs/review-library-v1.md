# Review Library v1 轻量体验增强

版本：v1.0

状态：In Progress

## 目标

让 Harmony 复盘库从「历史记录列表」变成「移动端复盘查看入口」。

## Harmony v1 能力

- 复盘库列表使用专用卡片，不再复用项目卡片。
- 卡片优先显示标题、是否成立、核心关系和当前卡点。
- 复盘时间和复盘人作为辅助信息展示。
- 复盘人为空时不显示脏占位。
- 当前卡点为空时不显示卡点行。
- 空状态明确提示当前还没有复盘记录，并提供「创建第一条复盘」入口。
- 删除确认继续说明只删除复盘库记录，不删除原照片或已导出的文件。

## 旧记录兼容

旧历史记录没有 `reviewerText` 时，列表不显示复盘人；筛选逻辑保持 v0 文档定义：这些记录仍出现在「全部复盘人」下，但不会出现在具体复盘人筛选项里，也不会匹配某个具体复盘人筛选。

## 边界

- 不新增标签、收藏、批注。
- 不做成长统计。
- 不做 AI 分析。
- 不修改 `review.json` 字段。
- 不修改 Mac `ReviewArchive` 或 sidecar 结构。
- 不删除原照片或已导出的 `review.json`。
