## 复提前检查

- 已明确反馈是否需要改代码、改截图、改隐私说明或仅补材料。
- 如涉及代码，已运行对应验证脚本和 `bash scripts/build_hap.sh`。
- 如涉及签名包，已重新确认 signed HAP 路径、文件大小和构建时间。
- 如涉及截图，已确认截图来自当前版本，不混入旧 UI。
- 如涉及权限，已核对 `entry/src/main/module.json5` 与华为后台权限说明一致。
- 已执行 `git status --short`，确认只提交本轮相关文件。
