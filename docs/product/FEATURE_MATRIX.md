# Skymap Product Feature Matrix

状态标记：

| 标记 | 含义 |
| --- | --- |
| 已支持 | 当前基线已经有可用产品能力。 |
| 体验增强 | 已形成闭环，本轮继续提升可用性和长期使用感。 |
| 规划中 | 产品方向明确，但当前能力不完整或未进入稳定闭环。 |
| 不支持 | 当前产品不提供该能力，也不能把它当成可用功能承诺。 |

## 能力矩阵

| 能力 | Mac | Harmony | 备注 |
| --- | --- | --- | --- |
| 摄影复盘 | 已支持 | 已支持 | Mac 支持摄影复盘卡与复盘内容导入；Harmony 支持手机端填写、阅读和导出复盘图。 |
| Review Library | 体验增强 | 体验增强 | Review Library 已从 v0 进入 v1 轻量体验增强：Mac 支持本地照片状态、缩略图和 Finder 定位；Harmony 支持移动端复盘库浏览、清晰列表和空状态引导。 |
| review.json | 已支持 | 已支持 | 本轮不改 `review.json` 字段。Review Library 的本地照片路径只属于 Mac 本地索引，不写入跨端交换文件。 |
| 本地照片定位 | 已支持 | 不支持 | Mac 仅在 Finder 中定位原图，不复制、不移动、不修改原图；Harmony 不承担桌面文件定位。 |
| 移动端复盘库浏览 | 规划中 | 已支持 | Harmony 复盘库是移动端查看入口，列表展示标题、判断、时间、复盘人、核心关系和当前卡点。 |
| 标签、收藏、批注 | 规划中 | 规划中 | 后续路线，不进入本轮 v1 轻量增强。 |
| 成长统计 | 规划中 | 规划中 | 不进入本轮。 |
| AI 分析 | 规划中 | 规划中 | 不进入本轮。 |

## v1 边界

Review Library v1 只提升查看、定位、回看和继续编辑体验；不做复杂标签系统，不做成长统计，不做 AI 分析，不修改 `review.json` 字段，不修改 Mac `ReviewArchive` 或 sidecar 结构。
