# Review Bundle 家庭存储浏览

日期：2026-06-28

本文定义 Mac 端“家庭存储复盘包浏览”能力。本能力只负责浏览家庭存储中的 v1 / v2 review bundle，并由用户手动选择单个复盘包导入本机复盘库。

## 一、功能定位

用户在家庭存储中已经有 HarmonyOS 导出的 `ReviewBundles` 目录。Mac 端可以选择这个根目录，扫描其中已有复盘包，显示候选列表，并复用现有导入逻辑导入单个 bundle。

典型目录：

```text
Skymap/
└── ReviewBundles/
    ├── 2026/
    │   └── 06/
    │       ├── review_20260627_212801_xxxxxxxx/
    │       └── review_20260628_153000_xxxxxxxx/
    └── review_20260701_101500_xxxxxxxx/
```

## 二、当前能力

- 选择家庭存储中的 `ReviewBundles` 根目录。
- 记住最近选择的根目录，下次打开时尝试恢复；目录不可访问时提示重新选择。
- 可清除已记住的根目录。
- 从根目录向下扫描最多三层。
- 支持 `ReviewBundles/YYYY/MM/review_xxx/`。
- 支持 `ReviewBundles/review_xxx/`。
- 读取每个候选 bundle 的协议文件并复用 Mac Reader 校验。
- 识别 v1 成品图复盘包与 v2 原图复盘包。
- 展示标题、类型、判断、时间、来源、导入状态和单个导入操作。
- 支持重新扫描；扫描失败时保留上一次成功结果，并提示检查权限或重新选择。
- 支持轻量筛选：全部、未导入、已导入、成品图包、原图包、异常。
- 默认按 bundle 创建时间、复盘时间、目录时间或文件修改时间倒序展示。
- 通过 `bundleId` 判断是否已导入。
- 已导入项可以定位本机复盘库记录。
- 未导入且校验通过的候选项可以单个导入。
- 异常项保留在列表中，可通过“异常”筛选查看，但不可导入。

## 三、v1 / v2 行为

- v1 成品图复盘包导入后进入本机复盘库，只读查看 `review-card.png`。
- v1 不显示“打开为复盘卡”。
- v1 不把成品图当作原图进入主编辑器。
- v2 原图复盘包导入后进入本机复盘库，可由用户显式打开为复盘卡。
- v2 打开为复盘卡时恢复原图、复盘内容和模板配置。
- v2 `exportedImages=[]` 合法，不要求成品图。

## 四、不包含

- 自动同步。
- 后台定时扫描。
- 批量导入。
- 双向同步。
- 远端删除。
- 写回家庭存储。
- 冲突自动合并。
- 云数据库。
- 修改 Review JSON 字段。
- 修改 manifest v1 / v2 协议。
- HarmonyOS 端实现变更。

## 五、权限与路径

Mac 使用系统目录选择器选择 `ReviewBundles` 根目录。客户端会保存系统授权目录记忆，并保留普通路径作为降级记忆。若下次打开时目录不可访问，用户需要重新选择。

当前能力不假设家庭存储路径永久有效，也不扫描用户整个磁盘。

界面只显示中间省略后的短路径，避免把 NAS、外接盘或本机目录的完整长路径暴露在主界面中。

## 六、扫描、筛选与异常

扫描只读取候选复盘包，不会导入、删除或写回家庭存储内容。

重新扫描行为：

- 使用当前已选择的 `ReviewBundles` 根目录重新扫描。
- 扫描中显示“正在扫描复盘包…”。
- 扫描成功后刷新候选列表，并重新计算已导入状态。
- 扫描失败时保留上一次成功结果，并提示“扫描失败，请检查目录权限或重新选择。”
- 扫描失败不会自动导入任何 bundle。

筛选项：

- 全部
- 未导入
- 已导入
- 成品图包
- 原图包
- 异常

异常项展示为普通用户可理解的原因，例如“文件不完整”“不支持的复盘包版本”或“无法导入”。界面不展示底层字段名、完整标识或调试信息；技术错误只保留在调试日志中。

## 七、已导入定位

已导入项显示“查看本地记录”。点击后切换到“本机复盘”，选中对应本机记录，并显示该记录的预览和复盘内容。

定位只按 `bundleId` 匹配本机复盘库记录，不会重复导入。如果本机记录已不存在，界面提示“本地记录不存在，请重新导入。”，并重新计算候选项导入状态。

## 八、仍不包含

- 自动同步。
- 后台扫描。
- 定时扫描。
- 批量导入。
- 远端删除。
- 写回家庭存储。
- 冲突合并。
- 云同步。
- HarmonyOS 端变更。

## 九、验收路径

1. 准备包含 v1 / v2 的 `ReviewBundles` 根目录。
2. 打开 Mac 复盘库。
3. 点击“浏览家庭存储复盘包”或切换到“家庭存储”。
4. 选择 `ReviewBundles` 根目录。
5. 预期扫描出 v1 / v2 候选列表。
6. 使用筛选查看未导入、已导入、成品图包、原图包和异常。
7. v1 显示“成品图复盘包”。
8. v2 显示“原图复盘包”。
9. 已导入项显示“已导入”，并可查看本地记录。
10. 点击 v1 导入后，本机复盘库显示只读成品图。
11. 点击 v2 导入后，本机复盘库显示原图，并可打开为复盘卡。
12. 点击“重新扫描”，列表刷新但不自动导入。
13. 关闭 App 再打开，确认目录可恢复；若目录不可访问，提示重新选择。
14. 选择错误目录时提示没有找到复盘包或检查目录权限。
